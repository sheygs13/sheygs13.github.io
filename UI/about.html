About Me

Hello There! 😃

My name is Olusegun Ekoh a.k.a sheygs. A Web Developer (JS, ...Web) from Lagos. I always thought my dream was to be an Electrical Engineer for one electrical company. It seemed like an appropriate dream given my B.Sc. in Physics with Electronics but I just kind of felt meh. I yearned to learn more, yearned to learn something different.

And that's when it happened; the spark I was missing ignited the moment I clicked play on my first Java tutorial video. Since then, I can't get enough. Front-end web development calls to all my passions; it incorporates creativity and problem solving and I'm free to break it to improve my code (of course on a git branch).😁

I love to apply responsive design principles and watch my web page shrink into mobile screens and still look amazing - very oddly satisfying😋

My specialities include:JavaScript, HTML5/CSS3, ReactJS, Git, NodeJS, PostgreSQL/SQL
